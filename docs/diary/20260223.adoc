== 新春雜筆

自從離職後，我花了將近3個月的時間發洩情緒。 +
想了很多想法，想要把在職時說離職後想做的事情給個遍。 +
實際上是作得少、想得多，當然身體和精神也恢復得多。 

我開始斷捨離，一一扔掉早就看不順眼的雜物。 +
和許久沒聯絡的朋友聚會，發現自己原來並不孤獨。 +
就不再責備自己是不是比別人差了。

終於得到一段我可以完全支配的空閒時間，
我最想做的事情就是創作一件作品，而現在，我可以選擇的載體更多了。 +
我感到滿足與驕傲，或許我所學的每樣技能不夠精深，但AI可以補足這一塊。 +
運用這些技能，我可以作出更能符合我喜好的作品。

或多或少還是會焦慮，於是我終於去了諮商，幸運地，我遇見的諮商師非常好。 +
兩次我離開諮商室，都是笑著走的。

新年過後，神奇地我更多地放下了那些不快的回憶。 +
依稀記得去年也是在開春後，我轉變心態，立下目標後積極地完成專案剩下的工作。 +
注意到這點，我又對自己多了一層認識。

我作了一點調整，把運動作為每天的首要任務。 +
餘下的時間再去分配做其他想做的事情。 +
新春有感而發，沒什麼重點，就想記錄一下此刻的心情。 +
願這一年對自己更溫柔、更主動地去追尋目標並活在當下。
